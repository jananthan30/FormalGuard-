# SOX Section 404 Compliance Mapping

## Scope

Maps FormalGuard properties to SOX Section 404 internal control requirements as they apply to hardware-level controls in financial transaction processing systems.

## Coverage

| Requirement | Description | Properties |
|-------------|-------------|-----------|
| 404.a.1 | Internal controls — hardware data integrity | 2 |
| 404.a.2 | Transaction processing controls | 3 |
| 404.b.1 | Access controls | 3 |
| 404.b.2 | Data integrity controls | 3 |
| 404.b.3 | Key management controls | 5 |

## Limitations

SOX Section 404 primarily addresses process and organizational controls. This mapping covers only the hardware-level controls that can be formally verified. Software controls, organizational policies, and audit processes are outside scope.
