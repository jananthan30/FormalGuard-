# SWIFT CSP Compliance Mapping

## Scope

Maps FormalGuard properties to SWIFT Customer Security Programme (CSP) controls focused on hardware-level security for SWIFT messaging infrastructure.

## Coverage

| Control | Description | Properties |
|---------|-------------|-----------|
| 1.1 | Internet access restriction / critical system protection | 2 |
| 1.2 | Operator credential control | 3 |
| 2.1 | Data exchange confidentiality and integrity | 6 |
| 2.2 | Operator session security | 3 |
| 2.6 | Message transmission protection | 2 |
| 4.1 | Security event logging and detection | 2 |
| 5.1 | Operator credential protection | 4 |

## Limitations

This mapping covers hardware-level controls only. SWIFT CSP has broader software, network, and process controls that are outside the scope of formal hardware verification.
